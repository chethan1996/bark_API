# bark_API
Dockerized bark API

Run below commands:

Step 1:
`docker-compose -f ./docker-compose.yml`

Step 2:
`docker-compose -f ./docker-compose.yml up`

Step 3:
visit [http://localhost:8000](http://localhost:8000)
